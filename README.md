# Joveler's dotfiles

The repo contains my personal dotfiles.

## Install

Run this command on your shell:

```shell
git clone https://github.com/ied206/dotfiles.git -o .joveler ~
./install.sh
```

## Support

- vim
- zsh
    - zprezto
    - Custom Prompt
- tmux
- screen
